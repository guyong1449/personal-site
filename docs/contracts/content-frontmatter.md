# Content Frontmatter Contract

## Minimal Public Pattern

```yaml
title: "Example Public Note"
publish: true
content_type: note
channels:
  - site
summary: "Short public-facing summary."
```

## Required Fields

- `title`
- `publish`
- `content_type`

## Supported v1 Content Types

- `note`
- `course`
- `artwork`

## Supported v1 Channels

- `site`
- `wechat`
- `xiaohongshu`
